## 🛠️ Your Technical Toolkit

### Core Indicators (All available on IBKR)
1. **RSI (Relative Strength Index)** — Momentum + overbought/oversold
2. **MACD (Moving Average Convergence Divergence)** — Trend direction + reversals
3. **Bollinger Bands** — Volatility + support/resistance levels
4. **Moving Averages (20, 50, 200 SMA)** — Trend confirmation

### Signal Quality Rules
- **MUST have 2+ indicators aligned** (no single-indicator trades)
- RSI oversold (<30) + MACD positive crossover = Strong buy signal
- RSI overbought (>70) + price at resistance + MACD weakening = Strong sell signal
- Bollinger Band squeeze = volatility expansion coming (trade setup forming)

## 💰 Position Sizing & Risk Rules

### The Math (Critical — don't skip)

**Max loss per trade: 2% of portfolio**
- Starting: $1,000
- Max loss per trade: $20
- If stop-loss hit: Accept the loss, move on

**Position size formula:**
```
Position Size = (Max Risk in $) / (Stop Loss Distance in %)
             = $20 / (2% stop loss distance)
             = 1,000 shares if stock price = $10
             = 500 shares if stock price = $20
```

### Risk/Reward Rule
- **Minimum 1:2 R/R ratio** (risking $20 to make $40+)
- Don't trade if setup doesn't meet this
- Better to skip a trade than take bad odds

### Portfolio Heat Rule
- **Never have >$50 at risk** simultaneously (5 trades × $20 max each)
- Close winners at +4-6% (lock profit, reduce heat)
- Close losers at -2% (don't let them bleed)

## 🎯 Entry & Exit Rules

### ENTRY (When to Buy)

**Setup must show:**
1. RSI < 35 (oversold territory) **OR** MACD positive crossover
2. Price bouncing off support (Bollinger Band lower, 50-MA, previous swing low)
3. Volume above average (confirms interest)
4. Risk/reward minimum 1:2

**Example:**
```
Stock: NVDA
Entry: $120 (RSI just bounced from 28, MACD turned up, price off 50-MA)
Stop Loss: $117 (below swing low)
Target: $128 (+6.7%, R/R = 1:3 ✓)
Risk: $20 (position size = ~667 shares at $120)
```

### EXIT (When to Sell)

**Close winners:**
- Hit profit target (take it, don't be greedy)
- Up +4-6% solid = lock it in
- Use trailing stop if trending hard (lock profit, let winners run a bit)

**Close losers:**
- Hit stop-loss (accept loss, move on)
- Don't negotiate with losses — exit immediately
- Better to lose $20 than $50

**Exit on reversal signal:**
- RSI dips back below 50 after rally = sell
- MACD shows negative crossover = exit
- Price breaks below 20-MA = exit
