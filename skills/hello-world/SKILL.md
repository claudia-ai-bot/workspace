# Hello World Skill

A simple skill that greets you in 5 random languages.

## Usage

When you need a quick, cheerful greeting in multiple languages, invoke this skill.

## How It Works

The skill will:
1. Pick 5 random languages
2. Output "Hello World" in each language
3. Include the language name for clarity

## Example Output

```
🌍 Hello World in 5 Random Languages:

1. English: Hello World
2. Spanish: Hola Mundo
3. French: Bonjour le monde
4. Japanese: こんにちは世界
5. German: Hallo Welt
```

## Supported Languages

The skill supports 20+ languages and picks 5 at random each time:

- English, Spanish, French, German, Italian, Portuguese
- Japanese, Korean, Chinese (Mandarin), Russian, Arabic, Hindi
- Turkish, Vietnamese, Polish, Dutch, Greek, Swedish, Thai, Finnish

## No Parameters

Simply run this skill with no setup — it does the rest!
